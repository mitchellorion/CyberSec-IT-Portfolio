# Flashcard CLI App

Hey! This is a simple flashcard app I made in Python that runs in the terminal.

- The text is bright neon green with electric blue borders (looks cool, right?)
- It shows a flickering “Loading modules...” message when you start it
- You can add new flashcards anytime through the command line
- Quiz yourself with random questions to practice what you’ve learned
- Your flashcards get saved in a file called `flashcards.json` so they’re there next time

---

## What you need

- Python 3 (make sure it’s installed!)
- A terminal that supports colors (most do nowadays)
